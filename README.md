# C++ Leaderboard System

A simple C++ program that creates a leaderboard based on player scores.

## Features
- Stores players using vector<pair<int,string>>
- Custom sorting using lambda function
- Tie-breaking based on alphabetical order
- Clean formatted leaderboard output

## Concepts Used
- C++ STL (vector, pair)
- Sorting with lambda comparator
- Object-Oriented Programming
- iomanip for formatted output

## Example Input
3
Aarav 95
Rohan 88
Priya 95

## Example Output
🏆 Leaderboard
1. Aarav  -> 95 pts
2. Priya  -> 95 pts
3. Rohan  -> 88 pts
